Sales Dashboard (Power BI)

This project presents a comprehensive Sales Dashboard developed using Power BI to analyze key business metrics such as sales performance, profit trends, and regional insights. The dashboard is designed to provide clear, interactive, and data-driven insights for better decision-making.

A preview of the dashboard is also included (dashboard_preview.png) for quick visualization.

Features

The dashboard includes analysis of overall sales and profit, region-wise and category-wise performance, and interactive filters that allow users to explore the data dynamically.

Files Included

sales dashboard.pbix – Main Power BI dashboard file

SuperStore_Sales_Dataset.csv – Dataset used in the dashboard

blue bg.jpg – Background image used in the report

dashboard_preview.png – Screenshot preview of the dashboard

How to Use

Download the .pbix file and open it using Power BI Desktop. If the dataset does not load properly, go to “Transform Data”, open “Data Source Settings”, and update the file path according to your system.

Tools and Technologies

Power BI was used for data visualization and dashboard creation, while the dataset is in CSV format.
